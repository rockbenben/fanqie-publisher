# 番茄作家 MD 批量上传工具

将本地 Markdown 文件批量上传到 [番茄作家](https://fanqienovel.com) 平台作为小说章节，支持存草稿、立即发布和定时发布。

## 功能特性

- **图形界面** — 全 GUI 操作，无需记命令
- **一键启动** — 首次运行自动安装依赖，开箱即用
- **批量上传** — 一次上传整个目录的 MD/TXT 文件，按文件名自然排序
- **三种发布模式** — 存草稿 / 立即发布 / 定时发布
- **智能排期** — 设置起始日期、每天章数和发布时间，自动计算排期
- **上次发布检测** — 自动获取最新定时发布时间和章节名，并建议下一天排期
- **章节管理快捷链接** — 一键在浏览器中打开对应作品的章节管理页
- **智能解析** — 从文件名或 `# 标题` 中自动提取章节号和标题
- **重复标题检测** — 自动检测并去重，避免同名章节限制
- **会话持久化** — 登录一次后保存状态，后续无需重复登录
- **错误截图** — 上传失败时自动保存截图，方便排查

## 快速开始

> 需要 [Python 3.10+](https://www.python.org/downloads/)，首次运行会自动安装其他依赖。
> Linux 用户还需安装 tkinter：`sudo apt install python3-tk`

- **Windows** — 双击 `run.bat`
- **macOS / Linux** — 终端运行 `bash run.sh`

首次运行会自动执行 `pip install` 和 `playwright install chromium`，之后直接启动 GUI。

## 使用流程

### 1. 登录

点击「登录番茄作家」，在弹出的浏览器中完成登录，然后点击弹窗「确定」保存会话。

### 2. 选择作品

登录成功后自动刷新作品列表。选择作品后会自动获取最新定时发布信息（章节名 + 时间），显示在发布模式区域。

点击「章节管理 ↗」可直接在浏览器中打开章节管理页面。

### 3. 选择章节目录

默认使用项目目录下的 `chapters/` 文件夹，也可点击「浏览...」选择其他目录。

「自动去重标题」默认勾选，避免番茄作家的同名章节限制。

### 4. 选择发布模式

| 模式         | 说明                                           |
| ------------ | ---------------------------------------------- |
| **存草稿**   | 上传后保存为草稿，可在后台逐章检查再手动发布   |
| **立即发布** | 上传后直接发布                                 |
| **定时发布** | 设置起始日期、每天章数和发布时间，自动排期发布 |

发布模式下可勾选「稿件使用了AI创作」，对应番茄作家的"是否使用AI"申报选项。如果作品使用了AI工具辅助创作，请如实勾选。

选择「定时发布」后，起始日期和时间会根据上次发布信息自动建议（上次日期 +1 天，保持相同时间）。

### 5. 预览与上传

章节预览区实时显示章节列表、字数统计和排期。确认无误后点击「开始上传」，运行日志区实时显示进度。

## MD 文件格式

每个 MD 或 TXT 文件对应一个章节：

```markdown
# 第 39 章 安定期

一周后。

周六下午两点。

林默和南俪并肩走在公园里。
```

- 第一个 `# 标题` 用于提取章节号和标题
- 如果没有 `# 标题`，则从文件名提取
- 正文中的 Markdown 格式标记会自动去除

**支持的文件名/标题格式：**

| 格式         | 示例                   | 章节号 | 标题                    |
| ------------ | ---------------------- | ------ | ----------------------- |
| 数字前缀     | `001_新的旅程.md`      | 001    | 新的旅程                |
| 第X章(数字)  | `第27章_重新开始.md`   | 27     | 重新开始                |
| 第X章(中文)  | `第十六章 发布会.txt`  | 16     | 发布会                  |
| chapter-N    | `chapter-039.md`       | 039    | _(从文件内 # 标题提取)_ |
| 纯文字       | `安定期.md`            | _(无)_ | 安定期                  |

## 配置文件

`config.json` 可调整默认行为：

```json
{
  "delay_between_chapters": 3,
  "headless": false
}
```

| 字段                     | 说明                 | 默认值  |
| ------------------------ | -------------------- | ------- |
| `delay_between_chapters` | 章节间等待秒数       | `3`     |
| `headless`               | 是否默认使用无头模式 | `false` |

## 常见问题

**登录状态过期了怎么办？**
点击「登录番茄作家」重新登录即可。

**上传失败怎么排查？**
失败的章节会自动截图保存在项目目录下（如 `error_0_chapter-039.png`），查看截图了解页面状态。

**可以断点续传吗？**
目前不支持。将已上传的 MD 文件移出目录，调整起始日期后重新上传。

**还能用命令行吗？**
可以。运行 `python fanqie_upload.py -h` 查看完整命令行参数。
